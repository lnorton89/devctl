    - `checkPortOccupied(port: number, host = 'localhost'): Promise<PortCheckResult>`:
      Uses `AbortController` with `setTimeout` for the 2s timeout (not socket.setTimeout).
      Attempts TCP connect via `net.createConnection({ host, port })`.
      On `connect` event → resolve `{ occupied: true }`.
      On `error` event where code is `ECONNREFUSED` or `ETIMEDOUT` → resolve `{ occupied: false }`.
      On other errors → resolve `{ occupied: false, error: error.message }`.
      On abort/timeout → resolve `{ occupied: true, error: 'Connection timed out.' }`.
      Clean up: socket.destroy() in all paths, clear timeout in all paths.

      Uses `AbortController` with `setTimeout` for the 2s timeout.
      Use the `http` or `https` module based on `new URL(url).protocol`.
      On response → check `statusCode >= 200 && statusCode < 300`. Resolve `{ healthy, statusCode }`.
      On `error` event → resolve `{ healthy: false, error: error.message }`.
      On abort/timeout → resolve `{ healthy: false, error: 'Health check timed out.' }`.

So for a running project, `port.occupied = true` means unhealthy (the port should be occupied by our process, but if we can't connect to it, that's unhealthy).     - **Add GET `/:id/health` endpoint** (after the existing `/:id/logs` route, before `return router`):
      ```
      router.get('/:id/health', async (req, res, next) => {
        try {
          const project = await loadProject(repository, req.params.id);
          const status = processManager.getStatus(project.id);

          // Only check health for running or unhealthy processes
          if (status.state !== 'running' && status.state !== 'unhealthy') {
            return res.json({ port: null, health: null });
          }

          const healthResult = await checkProjectHealth(project);

          // Transition state based on health check results:
          // - running + any check fails       → unhealthy
          // - unhealthy + all checks pass     → running
          const portUnhealthy = project.port != null && healthResult.port && !healthResult.port.occupied;
          const healthUnhealthy = project.healthUrl != null && healthResult.health && !healthResult.health.healthy;
          const isUnhealthy = portUnhealthy || healthUnhealthy;

          if (status.state === 'running' && isUnhealthy) {
            processManager.setState(project.id, 'unhealthy');
          } else if (status.state === 'unhealthy' && !isUnhealthy) {
            processManager.setState(project.id, 'running');
          }

          res.json(healthResult);
        } catch (error) {
          if (error instanceof ProjectNotFoundError) {
            return res.status(404).json({ message: error.message });
          }
          next(error);
        }
      });
      ```

    - **Modify POST `/:id/start`** to add pre-start port check:
      After loading the project but before script validation, add:

      if (project.port) {
        const portCheck = await checkPortOccupied(project.port);
