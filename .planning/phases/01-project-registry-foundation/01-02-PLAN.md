## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser/API input -> shared schema | Untrusted project configuration becomes typed registry data only after validation. |
| YAML file -> shared schema | Human-edited persisted YAML must be validated before use. |
| Registry data -> future command executor | Command strings and paths are stored but not executed. |
